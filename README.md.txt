# ✈️ Airport_Net

A Java GUI application that simulates an airport and flight network system.  
Users can search for airports by city and explore both direct and indirect flight connections between them.

---

## 🚀 Features

- Object-oriented design using Java
- Graphical User Interface with Swing
- View detailed information about airports
- Find direct & indirect flights between cities
- List airlines operating at each airport

---

## 🛠️ Technologies Used

- **Java 8+**
- **Swing** (for GUI)
- Object-Oriented Programming (OOP)
